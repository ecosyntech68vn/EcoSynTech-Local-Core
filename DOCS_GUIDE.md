## Cách đóng góp tài liệu

1. Tạo branch mới cho thay đổi tài liệu
2. Cập nhật nội dung theo template
3. Đảm bảo cập nhật "Last Updated"
4. Tạo Pull Request để review

## Tiêu chuẩn nội dung

- Sử dụng tiếng Việt làm ngôn ngữ chính
- Include "Last Updated" trong mỗi file
- Tuân thủ định dạng Markdown
